leetcode51 N皇后

无标签
发布日期:   2020-09-03
文章字数:   637

题目链接：https://leetcode-cn.com/problems/n-queens/

题目描述

n 皇后问题研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。

上图为 8 皇后问题的一种解法。

给定一个整数 n，返回所有不同的 n 皇后问题的解决方案。

每一种解法包含一个明确的 n 皇后问题的棋子放置方案，该方案中 ‘Q’ 和 ‘.’ 分别代表了皇后和空位。

示例

输入：4
输出：[
 [".Q..",  // 解法 1
  "...Q",
  "Q...",
  "..Q."],

 ["..Q.",  // 解法 2
  "Q...",
  "...Q",
  ".Q.."]
]
解释: 4 皇后问题存在两个不同的解法。

提示：

  • 皇后彼此不能相互攻击，也就是说：任何两个皇后都不能处于同一条横行、纵行或斜线上。

题目解析

可采用回溯算法，下图为 n=4 时的回溯路径。

思路：

从第一行开始，以第一列开始搜索，在第一行确定了 Q 之后，开始寻找第二行，下一个 Q 的位置必须不在之前 Q的同一列和斜线上面。确定之后在下一行找下一个满足的 Q，如果某行不满足，则退回上一行更改该行 Q 的列的位置（如果存在的话），存在则继续下一行寻找 Q，不存在则再退回上一行。

可以在每次搜索下一行的时候，基于该行 Q 的横纵坐标特点（x, y）来确定是否满足不在一列和斜线上。

同一列：y 相同

右斜线：x - y 相同

左斜线：x + y 相同

可利用三个集合来记录已经选好的 Q 的这三个值便于判断。

回溯代码如下：

class Solution(object):
    def solveNQueens(self, n):
        """
        :type n: int
        :rtype: List[List[str]]
        """
        board = [["." for i in range(n)] for j in range(n)]
        col = set()
        d1 = set()
        d2 = set()
        res = []

        def trans_board(board):
            res = []
            for item in board:
                res.append("".join(item))
            return res

        def backtrack(row):
            # 当且仅当搜索完最后一行，最后一行存在满足的Q
            if row == n:
                b = trans_board(board)
                res.append(b)
            else:
                for i in range(n):
                    if i in col or row - i in d1 or row + i in d2:
                        continue
                    col.add(i)
                    d1.add(row - i)
                    d2.add(row + i)
                    board[row][i] = "Q"
                    # 搜索下一行
                    backtrack(row + 1)
                    board[row][i] = "."
                    col.remove(i)
                    d1.remove(row - i)
                    d2.remove(row + i)

        backtrack(0)
        return res
